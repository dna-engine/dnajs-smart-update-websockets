Try it out:

1. Install Node.js from: https://nodejs.org
1. `git clone https://github.com/dna-engine/smart-update-websockets.git`
1. `cd smart-update-websockets`
1. `npm install`
1. `npm start`
1. In your web browser, go to: http://localhost:7777
